## üöÄ Getting Started

### Prerequisites
- Node.js 20.17+
- npm 10+
- Supabase account
- Vercel account (for deployment)

### Setup Steps
1. Clone repository
2. Navigate to `/apps/admin`
3. Install dependencies: `npm install`
4. Configure environment variables
5. Run development server: `npm run dev`
6. Access at `http://localhost:3001`
